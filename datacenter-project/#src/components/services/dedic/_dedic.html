Услуга "Выделенный сервер" ("Dedicated", "дедикейтед") - это аренда сервера, размещенного в нашем дата-центре с гарантированным доступом в сеть Интернет, бесперебойным питанием, кондиционированием, а так же круглосуточной технической поддержкой.

Dedicated server в переводе означает “выделенный сервер”. Это услуга центра обработки данных, которая предполагает передачу целого сервера в аренду одному клиенту. Потребитель, воспользовавшись услугой Dedicated service, получает всю его мощность для потребностей своего веб-ресурса, без необходимости делить ее с другими клиентами ЦОД. Поэтому данная услуга затребована для таких проектов, как крупные интернет-магазины, банки, страховые или иные финансовые организации, для которых недопустимо пользоваться сервером совместно с другими пользователями. Совместное использование одного сервера несколькими бизнес-пользователями ставит под угрозу сохранность и безопасность баз их данных.
