Возможности

Используйте возможности
почты HataMail

Почта
  • Привязывайте неограниченное количества своих доменов, создавайте неограниченное количество почтовых адресов
  • Делитесь доступом с коллегами к почтовым ящикам и папкам
  • Настраивайте внутреннюю рассылку, сбор почты из другого ящика, автоответчик и переадресацию
  • Применяйте специальные настройки антиспама, чтобы исходящие письма не попадали в спам
  • Архивируйте переписку и восстанавливайте утерянную
  • Используйте текстовый редактор,проверку орфографии, и другие функции для ускорения работы
  • Уведомления, поиск, статистика
    сделают работу с почтой удобной и эффективной
Облачное файловое хранилище Next Cloud
  • Загружайте файлы любого типа и размера через веб интерфейс, десктопное или мобильное приложение
  • Доступно для MacOS/iOS, Windows/Android
  • Обменивайтесь файлам и папками с коллегами и другими пользователями, предоставляя им доступ по ссылке
  • При необходимости устанавливайте срок действия и пароль на ссылку
  • Подключайте автоматическую загрузку фото и видео с телефона, что поможет освободить место на смартфоне
  • Просматривайте загруженные видео любого размера прямо с сервера с помощью встроенного плеера
  • Управляйте версиями файлов
Информация для администраторов
  • Возможность создания младших администраторов, которые могут управлять одним или несколькими из ваших доменов
  • Настройка любых, даже самых тонких, параметров почтового сервера под потребности вашей компании
  • Подключение любых почтовых клиентов, поддерживающих стандартные протоколы (outlook, thunderbird, android). Автонастройка через профиль на MacOS и iOS.
  • Постоянный мониторинг работы сервера и уведомления
  • Технический стек: Linux, PVE/PBS, Docker; Postfix, Rspamd, SOGo, Mailcow;
  • Поддержка DKIM, SPF, DMARC
  • Услуга базируется на 100% OpenSource ПО: MailCow, Icinga2, Ansible, Docker, Linux
